virtualbox 安装ubuntu server 16.04.6分辨率显示问题

发表于 2019-06-18 | 分类于 ubuntu | 次阅读

由于在virtualbox安装的是ubuntu server，没有GUI，无法通过virtualbox-设备-安装增强功能来进行窗口分辨率的调整。

参考链接

参考以上链接，通过命令行进行以下步骤：

  1. 通过VirtualBox environment来确定能够支持的分辨率。
    • 重启系统，并在进入界面时按住shift进入grub命令行，输入videoinfo来选择适合的分辨率（在我的虚拟机上显示为1152x864x32）
  2. 编辑/etc/default/grub来修改或创建GRUB_GFXMODE=1152x864x32，添加GRUB_GFXPAYLOAD_LINUX=1152x864x32
  3. 更新grub并重启upadate-grub2 && reboot

在重启的过程中就会成功将控制台的分辨率为所选分辨率。
