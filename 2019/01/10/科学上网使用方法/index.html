发表于 2019-01-10 | 分类于 折腾 | 次阅读

GitHub学生账号注册以及学生优惠领取

  1. 在GitHub使用校园邮箱注册账号
  2. GitHub Education获取学生优惠，包含多种优惠（AWS educate，digitalocean, gitkraken等）。
  3. 获取DigitalOcean $50 优惠券

在DigitalOcean上申请服务器

  1. 注册账号
    打开DigitalOcean官网：https://cloud.digitalocean.com ，右上角注册
  2. 验证邮箱
  3. 付款方式验证
    登陆到DigitalOcean后，右上角点击Account-Billing验证付款方式。

    有两种验证方法，一是绑定信用卡，二是通过paypal向账户充值5美元，建议使用第二种方法。（paypal官网：https://www.paypal.com/ ，绑定银行卡就行，与支付宝类似）

4.创建服务器

  • 点击Create Droplets
  • 选择系统和内存大小（推荐$5)
  • 选择服务器地区和额外选项
  • 选择密钥（关于如何配置SSH密钥请看下一个part）
  • 点击create就可完成创建。

配置SSH密钥

为了之后启动Digital Dropltes时，不使用邮件的IP地址和密码，通过设置SSH密钥访问服务器。SSH密钥是计算机和服务器之间的密钥对，如果服务器在您登录的计算机上看到匹配的密钥，则允许服务器进行连接。

MacOS/Linux

  1. 在本机创建密钥对并保存密钥


遇到文件保存等直接回车就可以，密钥生成完成之后。
公钥现在位于～/.ssh/id_rsa.pub，私钥（标识）位于～/.ssh/id_rsa中

  1. 复制密钥
    • 复制公钥：cat ~/.ssh/id_rsa.pub
    • 在DigitalOcean主页面左侧，Security中选择SSH keys-Add SSH keys。创建名称并粘贴复制的公钥。

Windows

参考DigitalOcean初始化–使用SSH Keys登录（Windows篇）

登陆远程服务器

MacOS/Linux


通过SSH密钥将直接登陆到服务器，如果未使用SSH密钥，则系统会提示输入密码，输入创建服务器时通过电子邮件发送的根密码。

Windows

参考DigitalOcean初始化–使用SSH Keys登录（Windows篇） ，通过PuTTY创建SSH会话并连接服务器。

安装Shadowsocks服务端和客户端

Shadowsocks服务端

  • 在服务器上执行Shadowsocks一键安装脚本（四合一）

  • 会提示选择Shadowsocks版本（我选择的是2 ShadowsocksR)

  • 接下来会有一些设定需要自己输入和完成

服务器端口：自己设定（如不设定，默认从 9000-19999 之间随机生成）
密码：自己设定（如不设定，默认为 teddysun.com）
加密方式：自己设定（如不设定，Python 和 libev 版默认为 aes-256-gcm，R 和 Go 版默认为 aes-256-cfb）
协议（protocol）：自己设定（如不设定，默认为 origin）（仅限 ShadowsocksR 版）
混淆（obfs）：自己设定（如不设定，默认为 plain）（仅限 ShadowsocksR 版）
备注：脚本默认创建单用户配置文件，如需配置多用户，请手动修改相应的配置文件后重启即可

  • 安装完成

    可以退出登陆啦。

在Ubuntu Server服务器上通过Proxychains科学上网

Server上没有图形化界面，Shadowsocks无法通过终端进行代理上网。

ProxyChains是Linux和其他Unices下的代理工具。 它可以使任何程序通过代理上网， 允许TCP和DNS通过代理隧道， 支持HTTP、 SOCKS4和SOCKS5类型的代理服务器， 并且可配置多个代理。 ProxyChains通过一个用户定义的代理列表强制连接指定的应用程序， 直接断开接收方和发送方的连接。

安装&配置

  • 选择apt方式安装proxychains

  • 更改配置文件sudo vim /etc/proxychains.conf
    在配置文件最后一行更改协议类型以及服务器地址和端口号。

使用方式

proxychains使用非常简单，在终端命令前加上proxychains
