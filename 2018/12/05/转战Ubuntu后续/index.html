转战Ubuntu后续

发表于 2018-12-05 | 分类于 折腾 | 次阅读

今天从Win10转用Ubuntu，提前考虑了一下，除了不能使用微信客户端和为知笔记客户端,哦，还有Itunes，没有其他对于Win10强烈需求。Linux之间交互更便利些，逐渐习惯吧，反正之前也是一直使用Xshell的，现在可以摆脱Xshell学生版四栏的限制了～


关于Ubuntu安装教程见链接

简单总结一下安装完Ubuntu16.04LTS之后遇到的问题：

中文输入法

  • 安装汉语语言包
  • 谷歌拼音输入法

  • 安装语言包以及切换键盘输入方式系统
    进入System Settings->Language Support,会自动提示安装语言包，安装完成后，将Keyboard input method system更改为fcitx

  • 重启电脑，可以观察到右上角输入法图标已经改变，fcitx输入法框架已经运行

  • 输入法配置
    • terminal输入：

会自动弹出窗口，然后左下角➕号选择合适的语言及输入法

成功～

proxychains使用Socks5代理

因为近期下载DARPA以及KDN的一些数据集，急需科学上网，偶尔Github也会大姨妈，所以直接就装个proxychains在终端使用代理吧。平时Chrome上使用SwitchyOmega，终端使用proxychains，应该能满足日常需求。
安装使用简单暴力：

  • 安装
  • 配置，在/etc/proxychains.conf最后一行把多余的注释掉，然后编辑
socks5 219.242.112.232 1080

  • 使用方法：在终端中，需要科学上网的命令前添加proxychains即可，特别方便。
    例如：下载DARPA 1998数据集的README （我在mac上也安装使用了proxychains，只不过mac上输入proxychains4)

成功～
